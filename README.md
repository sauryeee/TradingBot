# TradingBot

An AI-powered algorithmic trading platform built from scratch to learn software engineering, quantitative finance, machine learning, and artificial intelligence.

> **Project Status:** 🚧 In Development

## Objectives

* Learn professional software engineering practices.
* Build a modular algorithmic trading platform.
* Develop and evaluate trading strategies.
* Learn machine learning through real-world financial applications.
* Build an AI assistant for market analysis using a local large language model.

## Planned Features

* Historical market data collection
* Technical indicators (SMA, EMA, RSI, MACD)
* Strategy development
* Backtesting engine
* Machine learning models
* Interactive dashboard
* AI-powered market analysis
* Paper trading

## Technology Stack

* Python
* Git & GitHub
* Visual Studio Code
* Pandas
* Matplotlib
* Streamlit
* FastAPI
* PyTorch
* Ollama
* Docker (planned)
